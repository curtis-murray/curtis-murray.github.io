5.2.3 Constructing a topic structure

When conducting topic modelling in the context of a document-word network we get multiple matrices for each layer of depth in the hieararchy. One of these matrices is populated with \(p(\text{word} | \text{topic}, \text{level})\). From these matrices we can construct the hierarchical topic structure by noticing that words have non-zero probability for a particular topics if and only if they have a common ancestor at the given level.
